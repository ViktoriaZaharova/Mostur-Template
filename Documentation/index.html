Mostur – Travel & Tours HTML Template

Thank you for purchasing this template. This documentation will guide you through the installation and customization process.

1. Folder Structure

Mostur-Template/
 ├── Template/          ← main HTML template
 │     ├── index.html   ← homepage
 │     ├── about.html
 │     ├── contacts.html
 │     ├── booking.html
 │     ├── checkout.html
 │     ├── tour-page.html
 │     ├── tour-payment.html
 │     ├── certificates.html
 │     ├── reviews.html
 │     └── assets/
 │          ├── css/
 │          ├── js/
 │          ├── fonts/
 │          └── images/
 ├── Documentation/     ← this guide
 ├── Screenshots/       ← preview images
 └── README.txt

2. Installation

You don’t need special installation. Just upload the contents of the Template/ folder to your web server.

3. Editing HTML Files

Open any .html file in your code editor (VS Code, Sublime, etc.) and modify text, images, or structure as needed.

4. CSS & JavaScript

  • Main styles: assets/css/main.css
  • Bootstrap framework is included
  • Custom scripts: assets/js/

5. Fonts

Google Fonts are used. You can replace them in assets/css/fonts.css.

6. Images

All preview images are for demonstration only. Replace them with your own licensed images in assets/images/.

7. Support

If you have questions, please contact us via ThemeForest profile page.
