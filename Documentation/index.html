Mostur - Template Documentation

Thank you for purchasing the Mostur HTML Template. This guide will help you set up and customize the template.

Folder Structure

Mostur-Template/
│
├── Documentation/       ← this documentation
├── Screenshots/         ← preview images
├── Template/            ← main template files
│   ├── css/             ← CSS stylesheets
│   ├── js/              ← JavaScript files
│   ├── img/             ← images
│   ├── fonts/           ← webfonts (woff2)
│   └── *.html           ← HTML pages
├── readme.txt
├── changelog.txt
└── license.txt

How to Edit

  • Open Template/*.html files in any code editor (VS Code recommended).
  • Edit text, images, and links directly in HTML.
  • Modify styles in Template/css/style.css.
  • Replace images in Template/img/.

Fonts

All fonts are provided in .woff2 format. Example @font-face declaration:

@font-face {
  font-family: 'YourFont';
  src: url('../fonts/YourFont.woff2') format('woff2');
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}
